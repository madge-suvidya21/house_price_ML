# House Price Prediction using Machine Learning

## Project Overview

This project compares multiple machine learning algorithms on the Kaggle House Prices: Advanced Regression Techniques dataset.

## Algorithms Used

* Linear Regression
* Ridge Regression
* Lasso Regression
* Decision Tree Regressor
* Random Forest Regressor
* Gradient Boosting Regressor
* XGBoost Regressor

## Evaluation Metrics

* R² Score
* MAE (Mean Absolute Error)
* RMSE (Root Mean Squared Error)

## Visualizations

* Model Performance Comparison
* RMSE Comparison
* Actual vs Predicted Prices
* Feature Importance Analysis

## Technologies Used

* Python
* Pandas
* NumPy
* Matplotlib
* Scikit-learn
* XGBoost

## Results

Gradient Boosting and XGBoost achieved the highest prediction accuracy, outperforming traditional regression models by capturing complex feature relationships.

## Dataset

Kaggle: House Prices - Advanced Regression Techniques
